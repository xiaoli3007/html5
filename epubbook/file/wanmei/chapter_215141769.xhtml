第一千一百五十三章 白衣绝世

真的有这样一个人，跟石昊有关！

她不属于当世，不属于过去，踏在时间长河上，沐浴飞仙光雨，环绕无穷混沌气，俯视万古长空。

“真有人敢挡我之威，哈哈……哈哈哈！”远方，那推着日月星辰而行的高大身影，站在混沌中，发出大笑。

这一笑间，他周围的星辰簌簌坠落，一些大星更是直接炸开，化成烟花般的光，最后成为灰烬！

这种威势，无以伦比，天宇裂开，黑暗大裂缝扩散，黑洞一口接着一口的出现，漫天星辉暗淡。

石昊也是异常吃惊，这世间真的存在这样一个人，跟他有很不一般的关系？她居然就这么出现了！

他原以为，若真的有一个人也注定会是柳神，可现在看来并不是。

而这也让他心中发凉，感觉到了一阵凄怆，难道当日所看到的预言景象注定要成真吗？

当有一日，乾坤倾覆，岁月变迁，大世落幕，一切都不存在了，这世间只有他一个人孤立绝巅，独伴神道，孤独而无敌的寂寞，面对凋零的苍茫天地。

那个时候，只剩下他一个人了。一株柳树焦黑，成为背景，在他身后，那是何等的凄凉景象。

什么敌人，什么故人，什么亲人，什么朋友，爱恨情仇，一切不存在了。

隐约间，他还曾看到，他独自上路了，不知归路，不知前途。

这个女子不是柳神，是在那条路上相遇的人吗？

在这一刻，面对盘坐万古前的无上君主，生死攸关之际，有人来相救，石昊不知道是该哭还是该笑。眼中有泪，心中有悲，前路没有了故人，却是一个并不认识的女子。

如今的嬉笑，玩世不恭，也许终将会被沉重取代，因为很多事似已注定，他心中大恸，如果有朝一日，他不再是现在活跃的个性。变得深沉，独对大世，那将会何等的可悲，究竟都发生了怎样的事？

光想一想，他就觉得心中发凉，定然是一桩又一桩生离死别，看一个又一个熟悉的人上路，送他们离世而去。

分明有人来救，但此时石昊却这么的伤。因为他意识到许多事将会发生，心中压抑，感觉要窒息。

“我相信，未来可以改变。我看到的不会成真！”石昊低语。

那个女子很安静，踏在历史长河上，被混沌雾霭笼罩，修长的躯体寂静不动。唯有雪白的长裙随罡风而舞。

就是那一头乌黑的秀发也不动，自然披散在胸前与背后，她若一尊不朽的玉石雕刻而成。瑰美无比。

几人都想看清的她的容貌，但是却都做不到。

那鬼脸面具很特别，是比仙器还超凡，无法看透，即便是生有天目也不行，对它根本无效。

青铜面具，似哭非哭，似笑非笑，让人一望之下印象深刻。似有微笑，但却带着泪痕，明媚中亦有感伤。

而且下一刻那面具就变了，化成了一道指环，很古朴，浮华尽去，自然归真，在其身前沉浮，带着混沌光还有雾霭。

即便如此，也看不容那个女子的容貌，就是她的身体都很模糊，像是跳出了世界外，不再古今未来中。

接着，指环又变，化成了一口仙剑，一样无声，隐含杀意，含而不露，代表着这个女子不是真正的寂静不动，而有气吞山河意。

随后，仙剑又变，化成一个大道宝瓶，吞吐诸天星辰，缭绕着无尽仙气。

直至最后，它又化成面具，戴在了那个女子的脸上，寂静无声。

“以最平凡青铜为母器，凝练而成，成为仙器，又加入仙金精华，成就无上至宝，你在吗？”

远处，推动日月星辰而行、如同盖世帝君般的男子开口，不怒而威，乾坤星海都在隆隆颤动。

他盘坐于万古前，一念间，就可斩落无尽星斗！

女子平静，依旧是一句话也不说，没有在他身上留意，一双眸子清澈如水，此时有点点光泽渐渐发出，在看石昊。

无论跟石昊坐在一起的女子，还是那个恐怖的男子，都是一凛，好自信的女人，到现在竟只在关注那少年。

“哈哈……”男子大笑，威压世间，道：“也不知道多少年了，葬下了一世又一世，我都不知道还有谁敢这样面对我，你这个女人不错，很希望有一天使你常留在我身畔。”

他如一个帝王，看中了一个中意的女子般，说是自信也好，说是放肆也行，因为女子能显化在此，怎能轻慢？

果然，女子移开了目光，看向了他，并且双目中射出璀璨的光束，一瞬间气息完全不同了，如同一尊女帝，睥睨人间，威压天上地下，无远弗届。

这一刻，一股可怕的气息逆着时间长河直冲那男子而去，惊起滔天大浪，从天宇上卷落下一颗又一颗历史的星辰！

这种景象太过恐怖了，震撼人心。

仅是一股气息就如此，白衣女子绝代风华，傲视古今英雄。

“究竟过了多少世，几个纪元，才能出现一个你这样的女子。”在历史长河的上游，那个男人轻叹，被神光裹着，被混沌包围着。

而后，他的身体猛然爆发出了最为可怕的光芒，如无数颗太阳挤在一起，而后爆炸，最后更是化成了成千上万道最为璀璨的光束，如惊天彩虹，沿着长河而下，冲向那个女子。

很显然，他出手了，一刹那而已，就已经施展出了也不知道多少种大神通，一起迸发，斩杀那女子。

石昊心神摇动，就是坐在石桌对面的女人也是如此，天目睁开，紧张到了极点，没有人比她更清楚，出手的人多么的恐怖。

长河下游，那女子终于动了，右手扬起，洁白如玉，结了一个法印，而后向前拍去。

轰隆！

仅这一击而已，天地都为之颤栗了，诸天道则像是臣服了，虚空与大道都在簌簌颤抖。

没有话语，但是石昊还有其他两人都透过大道的轰鸣声，洞悉了四个字：万化仙诀！

他们明白了，这个女子手段超越了想象，已达不可思议之境，一种神通，降服大道，压制天地，乾坤在为其共鸣。

从那上游而来的成千上万道光束，代表了相应的一道又一道宝术，恐怖绝伦。

可是在这一刻，所有神通法则，所有光束都崩开了，都暗淡了，在那女子身前被化掉，成为虚无。

这是一个让人愕然的场面，那么多的大神通，在一刹那戛然而止，全部熄灭！

石昊还好说，因为不了解，不懂那个境界。可在石桌对面盘坐的女人就不同了，脸上露出惊骇的神色，那样的无上神通，接连发出，结果就被那白衣女子一掌拍的全部消失了？这是神话中的神话！

“真有这样的生灵，可以匹敌我？这种手段是你自创的吗，从未见过，倒也惊艳。”长河上游，那个男子踩着日月星辰，伴着混沌雾，这般说道。

从他这等人物口中说出惊艳两个字来，足以说明了一切。

无须细想，白衣女子绝对的惊才绝艳，不然的话，不会让那盘坐万古前、葬下了一世又一世的生灵这般感叹。

“轰！”

白衣女子动了，事实上她看着平静，但是一旦真正行动起来，却有绝世霸气，真正的俯视天地万灵，不将任何生灵看在眼中，傲绝古今。

她摘下了面具，抖手扔出，那面具化成覆天之幕，宏大无边，沿着历史长河而上，冲向那男子。

显然，她主动出击了，柔弱与被动根本不是她的本性！

“嗯？！”很明显，那男子大吃了一惊，仓促间，感觉要吃亏，快速应对，没有了早先的轻慢。

这青铜面具来的太快了，铺天盖地，压碎苍穹上的星斗，直接将那男子覆盖在下面，重重的而下！(未完待续。。)
