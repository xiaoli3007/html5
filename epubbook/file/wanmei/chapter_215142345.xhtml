第一千七百一十六章 覆灭的生命禁区

虚空中有太阴河与太阳河流交叉着，形成一个宏大的十字，壮观而惊人！

一条漆黑如墨，一条金黄璀璨；一条阴气重重，一条神圣炽烈；一个如同来自地狱，一个犹若发源神国。

这是两条完全不同的河流，都被大道符文笼罩着，蕴含着恐怖的力量，发出海啸般的声音，震耳欲聋。

这不是过去所看到的太阴河流与太阳河流，因为昔日所见，都是不纯净的，都只是凡河，而眼前这两条是根源，是祖河。

地上，也有两条，在那灰烬之下。那是如何形成的？

仔细推演，那真的是投影！

天地间，有真正的太阴与太阳，横亘宇宙中，它们投影向哪个界面，哪里就有太阴之气与太阳精气凝聚，最后跟凡河混在一起，形成水泽。

灰烬下的两条河，已经算是很纯净的了，但是同天穹上的两条相比，却什么都算不上。

轰隆！

石昊破开涟漪般的大道纹络，艰难的接近，到了虚空中，临近这里。

黄金狮子浑身的兽毛都根根炸立着，因为这里太恐怖了，那波动出来的气息等让它忍不住颤栗。

此地非同小可，让人敬畏，发自灵魂的悸动。

太阴与太阳，这是天地间最本源的两种规则，或许也可以称之为两种元素。

大道演化阴阳，对立而互补，便是五行元素等，都源自阴阳二气，这里是根。

在这太阴与太阳长河前，生灵自然会悸动，有无尽的感触，但凡修炼者到了这里都要灵魂颤抖。

石昊慢慢平静下来，渐渐适应这里的波动。

黄金狮子也不是一般的生灵。它一声低吼，也终于安静了很多，神魂不在发抖，跟着石昊一步一步临近。

换一个视角，他们看着两条交叉的河流，又觉得他们宁静而完美，平和而安然。

早先雷鸣般的声音，在这一刻竟然都消失了，那是一幅静美的画面，是世间最安谧之地。

怎么会如此？

这就是大道！

它变换无穷。不可捉摸。

可以说，太阴河流和太阳河流，是这天地间大道有形的体现，是最能说明规则与秩序的载体。

太阴与太阳之气，代表了阴阳，可以称之为规则中的母体，是超凡的力量。

现在，一人一兽有幸来到这里，当心中平静后。整片世界也都仿佛跟着安静了，在这里竟得到了不少好处。

最起码，他们对阴阳的理解一下子加深了。

这影响到了他们自身的路，石昊一声惊叹。体内隆隆作响，肉身中那些开启的门光芒大圣，绚烂夺目。

喀嚓！

直到远处一声轻响，两大强者才苏醒。从这种体悟中回过神来。

在十字河流附近，还有一些建筑，都残破了。也荒废了，刚才有一块石碑龟裂，惊醒了他们。

这是虚空中，没有土地，没有大陆，只有两条河流，它们是大道的一部分，也算是某种有形化身！

可是，当年却有生灵曾栖居这里，将之化成自家之地！

这是何其可怕的事？

这里曾为一个生命禁区，俯瞰九天十地！

至于地面上，那灰烬下的废墟，显然也是该族所留，但是可以料想，这里才是根基重地。

毫无疑问，有资格居住在这里的一定是核心成员，是最强的生灵。

只是，他们覆灭了！

而这种可怕的事，远胜过有种族可以在这里栖居这个事实。

究竟是怎样的一股力量可以将生命禁区推翻，将之铲除，看样子，当年一战十分的惨烈，不然的话地上怎会有那样的灰烬？

要知道，那灰烬可是诸神被焚烧形成的！

“门在哪里？”石昊寻找。

走在这片废墟中，他很谨慎。

因为，这是生命禁区的中心地，不知什么缘由覆灭。

很快，石昊打了个冷颤，而后黄金狮子则是颤抖，因为这里虽然建筑残破，并且不多，但令人压抑。

一块石碑龟裂，竟有不朽之气弥漫，让人惊悚。

显然，这里不可能有生命了，当年的人与事都被埋葬，生机等什么都不复存在。

可是，禁地生灵存在过的痕迹，依旧慑人心魄，尤其是这里，是核心人物的坐关地。

残破的建筑物，可以一眼望遍，什么都都没有留下，有的只是巨石、铜柱等，可是依旧让人寒毛倒竖。

不朽者栖居的地方，有他们散发的气场，多年过去，还没有散尽。

可以说，若非是石昊与黄金狮子足够强，一般的人根本走不到这里，会被压迫的瘫软在地上。

可以想象，如果这里没有破灭，即便无人防守，世间也没有几人可以走到这里，太过恐怖。

“门在哪？”黄金狮子也在找，它避开了废墟，踏在虚空中，在两条大河前徘徊。

就这样，他们寻找了很久，可是一点线索都没有，始终没有见到什么动静。

“太阴太阳，它们的尽头有一条古路，通向下界，可是，在哪里呢？”石昊自语。

这是他了解到的线索，如今来了，可就是找不到门路。

轰隆！

突然，他开始演化鲲鹏法，背后两只翅膀浮现，轻轻扇动，太阴与太阳之力弥漫，碰撞出了混沌气。

“是了，应该是在那里！”他醒悟。

在这里演化阴阳大道，让石昊醒悟，门最可能在什么地方？肯定是太阴河流与太阳河流的交汇处！

十字交叉地，有混沌气腾起，扩散开来，十分朦胧。

他到了近前，感受到了前所未有的压迫，十分小心，也很谨慎。

石昊投进去一件法器，没入河流交叉处，那里只冒出一些水花，而后混沌气汹涌，结果法器就被熔掉了，彻底毁灭。

这让黄金狮子倒吸冷气，不自禁向后倒退。

这是古路吗？真要跳进去的话，必死无疑啊。

“不对！”石昊蹙眉，他再次演化阴阳大道。

这一次，不仅是鲲鹏法，还有雷帝神通。

提到雷电，一般的人都认为那代表了毁灭，是上苍的惩罚，可是在当中却也蕴含着雷劫液这样的勃勃生机。

阴阳，生死，这是对立的，也是平和的而真实的。

石昊演化，通体发光，斑纹无数，一会儿为鲲，一会儿为大鹏，一会儿又成为雷电的化身，电弧密布。

轰！

一声剧震，天地都仿佛炸开了。

“嗷……”

黄金狮子鬓毛倒竖，事实上全身的毛都炸开了，像是刺猬一般，根根倒竖着。

它一声大叫，快速逃遁向远处。

因为，就在这一刻，太阴河流与太阳河流动了，改变了轨迹，一道从天空中垂落而下，一道横贯而过。

它们依旧交叉着，但是却改道了。

他们的交汇点，正是石昊那里。

黄金狮子简直不敢相信，这是幻觉还是真实的？

荒这是要沐浴太阴与太阳长河吗，要归入大道中？

无畏狮子看出来了，那两条河流大有讲究，是符文化成的，也是两种本源之力凝聚而成。

轰隆！

不过，糟糕的结局不曾出现，两条河流交汇处，出现一道门，很神秘，也很诡异，冒出混沌气，而石昊就在站在门外。

此时，门已经打开，隐约间可以看到，有一条古路，由灰褐色的石头铺成阶梯，通向远方未知处。

找到了，就是这里！

石昊心中激动，这一定就是他可以回到故乡的路！

他真的没有料到，需要以这种方式开启，必须精通阴阳大道，跟两条河流共鸣，才能开启出这条神秘的古路。

同时，他也有疑问，为何在太阴与太阳河流的交汇处有这么一条古径？

这是什么年代所留，是何人布置的？

再有，下界真的那么重要吗，竟然值得开凿出这样一条古路，保持贯通。(未完待续。)
